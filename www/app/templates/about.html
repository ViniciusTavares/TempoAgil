Sobre o Tempo Ágil

O TempoAgil nasceu com o propósito de adquirir experiência com o desenvolvimento híbrido, onde utilizamos o framework Ionic para desenvolver sua interface e grande parte de suas diretivas. Dados técnicos: Front End: HTML 5, CSS 3 e Javascript Back End: NodeJS, Express, MongoDB APIs de terceiros: Yahoo Weather API
